#  Telco Customer Churn Analysis & Prediction

##  Overview
End-to-end machine learning project on the Telco dataset to analyze and predict customer churn.

---

##  Problem
Telecom companies lose revenue when customers leave.  
This project identifies which customers are likely to churn.

---

##  Tools & Technologies
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  

---

##  Steps Performed
1. Data Cleaning  
2. Exploratory Data Analysis (EDA)  
3. Feature Engineering  
4. Machine Learning Models  
5. Evaluation  

---

##  Models Used
- Logistic Regression  
- Random Forest  

---

##  Evaluation Metrics
- Accuracy  
- ROC-AUC  
- Confusion Matrix  

---

##  Key Insights
- Month-to-month contracts have highest churn  
- Higher monthly charges increase churn probability  
- Tenure is strongly linked to customer retention  

---

##  Business Value
Helps telecom companies:
- Identify risky customers  
- Reduce churn rate  
- Improve retention strategies  

---

## Future Work
- Deploy model using Streamlit  
- Improve feature engineering  
- Hyperparameter tuning

---

##  Model Interpretation
The model shows that contract type, tenure, and monthly charges are the strongest predictors of churn.
